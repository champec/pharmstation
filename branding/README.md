# PharmStation Brand Guidelines

## Brand Identity

**Name**: PharmStation  
**Tagline**: "Your Digital Pharmacy Workstation"

PharmStation is a modern digital platform designed for UK pharmacies, combining compliance management with innovative AI-powered features. Our brand communicates trust, professionalism, and technological innovation in the healthcare space.

## Logo

The PharmStation logo features a mortar & pestle integrated with digital/data elements including:
- Bar chart imagery (representing data and analytics)
- A keyhole (representing security and compliance)
- An orbital ring (representing connectivity and modern technology)
- A sparkle (representing innovation and excellence)

### Logo Variants

1. **Light Version**: For use on dark backgrounds (Deep Blue, gradients, images)
2. **Dark Version**: For use on light backgrounds (Off-White, Pure White, light images)
3. **Icon Only**: The mortar & pestle symbol without text, for favicons and small applications

### Logo Typography

- "Pharm" is rendered in **Deep Blue** (`#257BB4`)
- "Station" is rendered in **Electric Cyan** (`#04B0FF`)

### Logo Usage Rules

**DO:**
- Maintain adequate clear space around the logo (minimum = height of the icon)
- Use the appropriate version (light/dark) for the background
- Maintain aspect ratio when resizing
- Use provided file formats (SVG preferred for web, PNG for documents)

**DON'T:**
- Distort, stretch, or rotate the logo
- Change the colors of logo elements
- Place the logo on busy backgrounds without sufficient contrast
- Use low-resolution versions
- Recreate or modify the logo

### Minimum Sizes
- Full logo: minimum width 120px (digital), 25mm (print)
- Icon only: minimum 24px × 24px (digital), 10mm × 10mm (print)

## Typography Recommendations

### Primary Font
**Inter** (Google Fonts) - For UI elements, body text, and general content
- Weights: 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold)
- Usage: Body text, buttons, forms, navigation

### Secondary Font
**Poppins** (Google Fonts) - For headings and display text
- Weights: 500 (Medium), 600 (Semibold), 700 (Bold)
- Usage: Headings, hero text, feature titles

### Monospace Font
**JetBrains Mono** - For code, register entries, and technical content
- Usage: CD register entries, logs, technical documentation

## Tone of Voice

### Professional yet Approachable
PharmStation speaks to pharmacy professionals with respect for their expertise while making technology accessible.

**Principles:**
- **Clear**: Use plain language; avoid unnecessary jargon
- **Confident**: We know pharmacy compliance and we've built the right solution
- **Helpful**: Guide users through processes without being patronizing
- **Trustworthy**: Acknowledge the serious nature of compliance work
- **Efficient**: Respect the user's time with concise, actionable content

**Examples:**
- ✅ "Sign in to your RP log in seconds"
- ❌ "Experience lightning-fast authentication flows"

- ✅ "Keep your CD register compliant and audit-ready"
- ❌ "Never worry about inspections again!"

### Content Guidelines

**Headlines**: Action-oriented, benefit-focused, under 60 characters  
**Body copy**: Scannable, use bullet points, short paragraphs  
**CTAs**: Clear action verbs (Sign in, Export, Create, View)  
**Error messages**: Explain what happened and how to fix it  
**Success messages**: Brief confirmation of action completed  

## Brand Applications

### Web Interface
- Use Off-White (`#E5F2F7`) for main backgrounds
- Deep Blue (`#257BB4`) for primary actions
- Electric Cyan (`#04B0FF`) for accents and highlights
- Cloud Blue (`#9FCADE`) for subtle glows on interactive elements

### Desktop Application
- Slightly darker palette option for reduced eye strain
- Consistent with web but optimized for longer sessions

### Mobile Application
- High contrast ratios for outdoor visibility
- Touch targets minimum 44px
- Simplified color usage for smaller screens

### Print Materials
- CD Register printouts: Clean, professional, high contrast
- RP Log exports: Clear typography, structured layout
- Certificates: Professional styling with subtle brand elements

## Accessibility

All brand applications must meet WCAG 2.1 Level AA standards:

- **Color Contrast**: Minimum 4.5:1 for normal text, 3:1 for large text
- **Not Color Alone**: Never rely solely on color to convey information
- **Focus Indicators**: Clear, visible focus states for all interactive elements
- **Alt Text**: All logo uses must include appropriate alt text

### Tested Color Combinations

**Text on Backgrounds** (all meet WCAG AA):
- Deep Blue text on Off-White: ✅ 7.2:1
- Deep Blue text on Pure White: ✅ 7.4:1
- Pure White text on Deep Blue: ✅ 7.4:1
- Pure White text on Mid Blue: ✅ 5.8:1
